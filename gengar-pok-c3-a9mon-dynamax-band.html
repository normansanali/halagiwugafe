Release information

This tag was packaged with the Ash's Strongest Meza Set Pokémon Dynamax Band, released on November 25, 2021. The set also included promotional Pikachu and Lucario tags.

Trivia

Origin

This Gengar is based on Ash's Gengar as it appears in Pokémon Journeys: The Series. An image of Ash will appear when Gengar attacks in-game. Shadow Ball is a move that Gengar can learn in the games. G-Max Terror is a G-Max Move Gengar can use while Gigantamaxed.
